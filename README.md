  <blockquote align="start">
    <b>홈 화면:</b> 사용자가 서비스에 입장하면 ‘카테고리’ 배너와 종류별 카테고리를 볼 수 있습니다. 하단에는 홈페이지, 검색, 카테고리, 퀴즈로 구성된 내비게이션 바가 존재합니다. 
    <br/><br/>
    <b>검색:</b> 검색 아이콘을 눌러 칵테일을 찾아볼 수 있습니다. 검색 중에 원하는 킥테일이 검색어로 뜨게 되는데, 이때 원하는 항목을 눌러 칵테일로 이동할 수 있다.
  </blockquote>

    <br/><br/>
    <b>퀴즈:</b> 퀴즈에서는 칵테일을 잘 몰라서 추천 받고싶은 사람들을 위한 퀴즈가 진행됩니다.
